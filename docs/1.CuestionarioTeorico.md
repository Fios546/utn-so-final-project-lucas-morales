### **✍️ Respuesta Corta**

**1️⃣ ¿Cuál es la diferencia entre un contenedor y una imagen en Docker?**

Una imagen es una plantilla que contiene la aplicación y sus dependencias. Mientras que un contenedor es una instancia en ejecución de esa imagen.

**2️⃣ ¿Para qué se utilizan volúmenes en Docker?**
Los volúmenes se utilizan para almacenar datos de forma persistente, de modo que no se pierdan al eliminar o recrear un contenedor.

**3️⃣ ¿Qué hace la instrucción `EXPOSE` en un Dockerfile?**
Indica qué puerto utiliza la aplicacion dentro del contenedor y puede servir como documentación para quienes usan la imagen.

**4️⃣ Escribe un beneficio de usar Docker en tus proyectos.**
Una ventaja es que permite que la aplicacion se ejecute de forma consistente en diferentes entornos, evitando problemas de dependencias o de configuración.

**5️⃣ ¿Cuál es el propósito de usar Nginx en tu proyecto Docker?**
Se utiliza para servir los archivos estáticos del frontend y actuar como un proxy inverso, redirigiendo las solicitudes de la API al contenedor correspondiente.
